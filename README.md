# ChatGPT / Claude / SuperGrok会员充值

这是一个 GitHub Pages 静态项目，用于说明 ChatGPT、Claude、SuperGrok 会员的官方订阅、付款、账单管理和账号安全注意事项。

## 项目边界

本项目仅提供官方渠道说明：

- 不提供代充服务
- 不收集 OpenAI / ChatGPT 账号密码
- 不代替用户登录账号
- 不提供账号共享
- 不协助绕过官方支付限制或地区规则
- 不处理来源不明的支付、礼品卡或充值请求

## 推荐说明

用户应通过 ChatGPT 官方网站或官方移动端应用自行订阅会员，并在官方账单页面管理付款方式、订阅状态、发票和取消续费。

- `index.html` - 会员订阅说明页
