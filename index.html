Billing

官方充值与账单管理步骤

  1. 登录 ChatGPT 打开 ChatGPT 官方网站，使用自己的账号登录。
  2. 进入升级或账单页面 在个人菜单中进入升级方案、设置、账单或 My Plan 页面。
  3. 选择会员方案 根据官方页面展示选择 Plus、Pro、Go 或其他可用方案。
  4. 使用官方结账渠道付款 付款方式、币种、税费和发票信息以官方结账页和账单页为准。
  5. 保存账单记录 订阅成功后，可在账单设置中查看付款方式、订阅状态和历史账单。

Safety

账号与付款安全提醒

  • 只在 ChatGPT 官方网站或官方移动端应用中完成订阅。
  • 不要把 OpenAI 账号密码、验证码、恢复码或付款信息发给第三方。
  • 不要购买来源不明的低价会员、共享账号或所谓内部充值。
  • 遇到未知扣费、账单异常或订阅问题，应通过 OpenAI 官方帮助中心处理。

FAQ

可以代充 ChatGPT 会员吗？

不建议。为了账号和付款安全，会员订阅应由用户本人通过官方渠道完成。

ChatGPT 会员和 API 账单是同一个吗？

不是。ChatGPT 会员订阅和 OpenAI API 使用通常分别计费，管理入口也不同。

如何取消订阅？

在 ChatGPT 设置或账单页面中管理当前方案，按官方页面提示取消或变更订阅。
